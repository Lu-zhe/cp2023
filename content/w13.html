w13

在近端建立 OpenSSH 格式的 public key, putty 格式的 private 並且配合 putty session 讓使用者可以將倉儲 git clone 到近端維護

檔案位置:tinyc裡的data，需有portablegit PuTTY Python311三個檔案

注意需在符合上述條件下開啟start_ipv6.bat，因為在虛擬主機(Y)所做的檔案更改不會影響自己本身的USB磁碟機

也會影響後續在檔案傳輸和指示字元讀取倉儲檔案上出現問題

在執行步驟上，使用puttygen.exe定位主機位置並獲得key，如圖(鼠標在其範圍內不斷移動直到綠條跑完)

將key(圖上藍色框選的亂碼)複製新增到自己倉儲帳號(設定-SSH keys)，同時也要點圖上的Save private key

儲存到tinyc-data-home(名稱學號)

在命令字元打上reedit，尋找到剛剛處存的key整葛目錄檔後右鍵匯出

到Y槽輸入git clone --recurse-submodules https://github.com/zhe41223118/cp2023.git

(上述藍色網址那段是自己個人的倉儲網址)

將倉儲資料複製到USB，之後cd進cp2023，輸入cms 就可開啟近端網站進行維護了
